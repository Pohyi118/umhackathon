## 🎮 Interactive Experience

### Mode 1: Hiring ROI (Default)
**What it does**: Validates if a hire's expected productivity justifies the cost

**User Interaction**:
```
1. Sees "Expected Monthly Revenue" slider
2. Drags to set projection (RM 3,955 to RM 39,550)
3. Reads the Verdict in real-time:
   🟢 Profitable Hire (≥100% coverage)
   🔴 High-Risk Hire (<100% coverage)
4. Checks progress bar to see visual gap
5. Reads one-sentence summary to understand the implication
```

**Example**: 
- True Cost = RM 3,955/month
- Target = RM 11,865 (3x breakeven)
- User sets Expected Revenue = RM 15,000
- **Result**: ✅ 126% coverage → Profitable Hire 🟢

---

### Mode 2: Severance Risk (Alternative View)
**What it does**: Forecasts termination costs for different employment durations

**User Interaction**:
```
1. Sees "Years of Service" slider
2. Drags to scenario-test different employment lengths (1-30 years)
3. Reads severance breakdown:
   • Formula: 10/15/20 days per year (EA1955)
   • Daily Wage: salary ÷ 26
   • Total Cost: how much termination would cost
4. Sees Verdict with recovery timeline:
   🟡 X.X Months to Recover Severance
5. Reads summary explaining the financial impact
```

**Example**:
- Base Salary = RM 3,500
- Years of Service = 5 years
- Daily Wage = RM 134.62
- Severance = 15 days/year × 5 years × RM 134.62 = RM 10,096
- **Result**: 🟡 2.6 months recovery (10,096 ÷ 3,955 = 2.55 months)

## 🔄 Real-Time Reactivity

### When Parent Salary Changes
```javascript
// User moves salary slider in StatutoryView from RM 3,500 to RM 5,000

// StatutoryView recalculates:
salary = 5000
epfEmployer = 5000 × 0.13 = 650 RM
socsoEmployer = min(5000, 6000) × 0.0175 = 87.50 RM
eisEmployer = min(5000, 6000) × 0.002 = 10 RM
totalEmployerCost = 5000 + 650 + 87.50 + 10 = 5,747.50 RM

// StrategicInsightPanel receives new prop:
<StrategicInsightPanel 
  baseSalary={5000}         // ← UPDATED
  trueCost={5747.50}        // ← UPDATED
/>

// Panel recalculates:
targetBreakeven = 5747.50 × 3 = 17,242.50 RM
severanceCost = 15 × yearsOfService × (5000/26) = changes!
recoveryMonths = severanceCost / 5747.50 = changes!

// UI updates instantly—sliders stay where user left them,
// but verdicts/calculations reflect new salary
```

## 📊 Calculation Formulas Reference

### Hiring ROI Mode
```
Target Breakeven = True Cost × 3
Coverage % = (Expected Revenue ÷ Target Breakeven) × 100
Status = Coverage >= 100% ? "Profitable" : "High-Risk"
```

### Severance Risk Mode
```
Days Per Year = tenure < 2 ? 10 : tenure < 5 ? 15 : 20
Daily Wage = Base Salary ÷ 26
Severance Cost = Days Per Year × Years × Daily Wage
Recovery Months = Severance Cost ÷ True Cost
```
